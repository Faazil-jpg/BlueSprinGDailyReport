One-time Firebase setup required

This app stores data in Firebase (free) so all your staff share the same database. Follow these steps once, then you're done forever.

1
Create a free Firebase project
Go to console.firebase.google.com → click Add project → give it any name (e.g. bluespring-reports) → click through the wizard.
2
Register a Web App & copy the config
In your project → click the </> (Web) icon → give it a nickname → click Register app → copy the firebaseConfig object shown.
3
Enable Firestore Database
In Firebase Console → Build → Firestore Database → Create database → choose Start in test mode → pick any location → Done.
4
Paste your config below
Fill in each field from the firebaseConfig object you copied above.

Config is saved in your browser. You only do this once per device.
